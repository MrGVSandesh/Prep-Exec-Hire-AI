# Prep-Exec-Hire-AI
Prep-Exec-Hire-AI is an advanced AI-powered interview preparation and evaluation platform designed to simulate real-world technical and HR interview experiences using NLP and Transformer-based deep learning models.
# 🧠 Technologies Used

- Python
- Streamlit
- NLP
- Transformers
- Sentence-BERT
- Scikit-learn
- PDFPlumber
- Machine Learning
- Deep Learning

# 🎯 Key Functionalities

## 🔹 Resume Upload
Upload PDF resumes directly into the platform.

## 🔹 Skill Extraction
The system detects:
- Python
- Machine Learning
- SQL
- NLP
- Deep Learning
- React
- and more...

## 🔹 AI Question Generation
Questions are dynamically generated based on detected skills.

Example:
```text
Explain Python decorators.
What is overfitting?
Explain transformers in NLP.
```

## 🔹 Semantic Evaluation
The system compares:
```text
User Answer ↔ Expected Semantic Meaning
```

using Sentence-BERT embeddings and cosine similarity.

## 🔹 AI Feedback
Examples:
- Excellent answer
- Improve technical explanation
- Add more practical examples
